2. COMPONENTES CREADOS
═══════════════════════════════════════════════════════════════════════════════

📦 MODELOS (src/models/)
─────────────────────────────────────────────────────────────────────────────

User.js
  Propiedades:
    • id: identificador único
    • nombre: nombre del usuario
    • email: correo electrónico
    • contraseña: contraseña encriptada
    • rol: 'admin' o 'usuario'
    
  Métodos:
    • validar(): valida integridad de datos
    • toJSON(): convierte a JSON

Medicamento.js
  Propiedades:
    • id: identificador único
    • nombre: nombre del medicamento
    • descripcion: descripción
    • precio: precio en USD
    • stock: cantidad disponible
    • categoria: categoría del medicamento
    
  Métodos:
    • validar(): valida datos
    • hayStock(cantidad): verifica disponibilidad
    • reducirStock(cantidad): reduce el stock
    • aumentarStock(cantidad): aumenta el stock
    • toJSON(): convierte a JSON


🎮 CONTROLADORES (src/controllers/)
─────────────────────────────────────────────────────────────────────────────

AuthController.js
  Métodos:
    • login(email, contraseña): autentica usuario
    • registro(nombre, email, contraseña): registra nuevo usuario
    • logout(): cierra sesión
    • obtenerUsuarioActual(): obtiene usuario autenticado
    • estaAutenticado(): verifica sesión activa
    • obtenerUsuarios(): obtiene lista de usuarios

MedicamentoController.js
  Métodos:
    • obtenerTodos(): obtiene todos los medicamentos
    • obtenerPorId(id): obtiene medicamento específico
    • buscar(termino): busca por nombre o categoría
    • crear(nombre, descripción, precio, stock, categoria): crea nuevo
    • actualizar(id, datos): actualiza medicamento existente
    • eliminar(id): elimina medicamento
    • obtenerPorCategoria(categoria): filtra por categoría
    • obtenerStockBajo(limite): medicamentos con stock crítico
